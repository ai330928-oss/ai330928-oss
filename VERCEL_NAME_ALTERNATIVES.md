# Vercel 프로젝트 이름 대안

## 현재 상황
`multisports-manager` 이름이 이미 사용 중입니다.

## 해결 방법

### 방법 1: 다른 이름 사용 (권장)

다음 이름 중 하나를 선택하세요:

1. **multisports-manager-game**
   - URL: `multisports-manager-game.vercel.app`

2. **multisports-sim**
   - URL: `multisports-sim.vercel.app`

3. **sports-manager-pro**
   - URL: `sports-manager-pro.vercel.app`

4. **multisports-app**
   - URL: `multisports-app.vercel.app`

5. **team-manager-game**
   - URL: `team-manager-game.vercel.app`

6. **squad-manager-2024**
   - URL: `squad-manager-2024.vercel.app`

7. **multisports-oss** (GitHub 저장소 이름과 유사)
   - URL: `multisports-oss.vercel.app`

### 방법 2: 기존 프로젝트 사용

만약 `multisports-manager`가 본인의 기존 프로젝트라면:
1. Vercel 대시보드에서 기존 프로젝트 선택
2. Settings → Git → Connect Git Repository
3. `ai330928-oss/ai330928-oss` 저장소 연결

### 방법 3: 기존 프로젝트 삭제 후 재생성

1. Vercel 대시보드 → 기존 `multisports-manager` 프로젝트
2. Settings → Delete Project
3. 새로 프로젝트 생성

## 추천

**multisports-manager-game** 또는 **multisports-sim**을 추천합니다.

## 참고

- 프로젝트 이름은 나중에 변경 가능하지만, URL은 변경되지 않습니다
- PWA 앱 이름은 별도로 설정됩니다 (vite.config.js)
- 사용자에게 보이는 이름과 프로젝트 이름은 다를 수 있습니다
